The Coherence Principle üéØ

Align all learning activities to a learning goal.

For example, if a learning goal is to use evidence in writing, a learning activity would be to write an essay that requires evidence. We wouldn't want to ask students to complete a matching quiz because that wouldn't achieve the learning goal of writing.
