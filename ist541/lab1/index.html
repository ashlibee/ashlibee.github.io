How to Change Your Google Chrome Background Image
on a Windows Device

By Ashli Bumgardner


Step 1: Launch Google Chrome

In order to update the Google Chrome background, it must be updated within the Chrome browser. It cannot be updated from a different browser.


Step 2: Select "Customize Chrome"

The "Customize Chrome" button is located in the lower-right corner in any newly opened Chrome window.


Step 3: Choose "Upload from device"

This is the first option. There are other stock image options that can be selected, but to upload your own image saved on your device, select "Upload from device."


If you're interested in browsing free images available for reuse, visit Unsplash.com for openly licensed photos.


Step 4: Select the image file

You will be brought to a file explorer. You will want to open the folder where you've saved the image you want to use as your background. Once you've opened that folder, click on the image to select the file.

To the right is an example of an image file. Yours will have a different file name.


Step 5: Select "Open"

Select the "Open" button from the lower right of the file explorer.

As soon as you select "Open," you should see your updated background on Chrome.
